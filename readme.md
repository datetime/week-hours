```
npm i --save week-hours
npm test
```


## API
> For more use-cases see the [tests](./test.js)

### [weekHours](./index.js#L28)
> Get the number of hours in a week.

- `return` **{Number}**

**Example:**

```js
var weekHours = require('week-hours');

weekHours();
//=> 168
```
